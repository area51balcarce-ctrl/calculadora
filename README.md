# SERVICIOS INTEGRALES - Calculadora interna de cupo

Fórmula exacta:

Resultado X = Hab. c/Ap. - IPS - IOMA

Base 75% = Resultado X * 0.75

Cupo final = Base 75% - suma de todos los descuentos debajo de IOMA

Regla clave: todo importe que aparezca por debajo de IOMA se resta después de calcular el 75%, hasta llegar a neto, totales, firma, banco u otro bloque de cierre.
